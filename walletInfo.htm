How to create a MetaMask Wallet

To get paid in USDC, you need a crypto wallet for the Avalanche blockchain.
You will need an Ethereum compatible wallet, which can interact with WalletConnect.

Caution: The Coinbase mobile wallet is currently not supported.


Install and Configure Metamask


We recommend using MetaMask, available as a browser extension, for Android and iOS.

You can create new Metamask wallet with the following steps:

  1. Install MetaMask for Desktop or mobile.
  2. Open the app/extension and go to new wallet. Enter a strong password and confirm it. MetaMask cannot recover this password, so you need to remember it.
  3. Secure your wallet, by storing a unique seed phrase. A seed phrase is unique to every account and it provides the only way to recover your wallet by importing your account using the seed phrase. Neither loose nor share your seed phrase with anyone.
  4. Enter the seed phrase and click the "Complete Backup" button.
  5. Your account has been successfully created. The wallet has a public address and a private address. Never share your private address with anyone, otherwise your assets might get stolen.

If you are using the browser extension, make sure NOT to start the experiment in Incognito Mode.


Switch to the Avalanche network


By default, Metamask is connected to the Ethereum network, and you need to take the following steps to add the Avalanche network C-Chain.

An explanation on how to do it can be found here:

Check your balance in USDC


If you would like to see the USDC token balance on your wallet you have to add them first to your wallet:

  1. Open MetaMask
  2. Click on "Token importieren"
  3. Insert the Token Address: 0xB97EF9Ef8734C71904D8002F8b6Bc66Dd9c48a6E and the Token Symbol: USDC
  4. Click Import ("Importieren" in German)

Technical information about the USDC token (including all transactions) can be found here: https://snowtrace.io/token/0xB97EF9Ef8734C71904D8002F8b6Bc66Dd9c48a6E
